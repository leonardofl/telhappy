Para utilizar o Telhappy:

1. Descompate o zip em qualquer lugar

2. Se seu sistema operacional estiver com o Java instalado e tudo configurado certinho, basta um duplo-clique em arquivo jar

3. Se sua máquina estiver com Java, mas o programa não abrir com duplo-clique, você pode executar a seguinte linha de comando: "java -jar telhappy.jar"
