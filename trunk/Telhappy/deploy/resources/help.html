Help!

TellHappy é um sistema que que auxilia a atribuir responsáveis por ligações não identificadas em uma república.
Para isso deve-se partir do seguinte princípio: se alguma vez na vida alguém P ligou para um número N, sendo esse número N registrado no sistema, e em seguida alguém X ligar para esse número sem anotá-lo devidamente na caderneta de telefones, fazemos a atribuição X = P.

A versão 2.0 conta agora com o sistema de conta automática para contas da NET ou arquivos CSV, que procura fazer os passos anteriores de forma bem mais automatizada

Etapas do processo:

0. Os habitantes da república já devem ter anotado todas as suas ligações em uma caderneta específica com os seguintes campos: telefone, data e usuário;
1. Iniciar o TelHappy System; se o programa não iniciar com um duplo clique no jar, é possível inicia-lo através da seguinte linha de comando: "java -jar telhappy.jar";
2. Incluir os números das ligações efetuadas no período para o TelHappy System (opção "Incluir telefones");
3. Efetuar a "conta automática" para checar quanto cada morador deve pagar. Mais sobre a conta automática na seção abaixo.
Caso você não possa efetuar a conta automática, ainda é possível o seguinte fluxo alternativo:
3b. Seguindo a conta telefônica, procurar os números das ligações realizadas no sistema (opção "Encontrar telefoneas não identificados");
4b. Somar os valores adquiridos para cada pessoa;

Na etapa 3b devemos tomar as seguintes precauções:

a) Analisando a caderneta marcar primeiro os telefones de utilidade pública (i.e: em nome da república);
b) Caso o sistema acuse um telefone de um habitante ou um telefone com mais de um possível usuário, deve-se recorrer à caderneta para identificar o responsável. Caso a caderneta não responda a questão, a ligação é dividida em comum pelos moradores (ou alternativamente se instaura um inquérito);

Conta automática

Caso você seja assinante da NET e receber a conta digitalizada em PDF você pode escolher a opção NET.
O processo passa por duas etapas, converter o PDF para txt e depois efetuar a divisão, sendo que essa primeira etapa é delegada para o programa pdftotext (versão 0.12.0).
O pdftotext já vem instalado no Ubuntu; não sei se dá pra instalar no Windows, mas se for possível, para que tudo funcione é preciso que a chamada para o pdftotext esteja no path do sistema (i.e.: você de ser capaz de invocar o pdftotext a partir de qualquer diretório no prompt do DOS).

Caso você não tenha o PDF da NET ou não consiga instalar o pdftotext, você pode construir um arquivo CSV com as seguintes colunas: telefone, data e valor.
Exemplo de arquivo CSV:

1231224333, 12/07/10, 1.40
114544523, 13/07/10, 2.33
117692318, 19/07/10, 6.82
